更新方法
1. 念のためアプリの「データ」から完全バックアップを保存
2. このフォルダの全ファイルをGitHubのdashboard-v4へ上書きアップロード
3. Commit changesを押す
4. 1〜2分後にSafariで再読み込み

キャッシュが残る場合はSafariを閉じて再度開いてください。
