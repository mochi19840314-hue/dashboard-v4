【今回の変更】
1. 年間画面のKPIカードをApple風デザインに刷新
2. 年間利益を「年間売上－年間支出」で自動計算
3. 年間利益率・月平均利益を追加
4. 当月支出が旧形式で保存されていても利益へ反映
5. 年間グラフを滑らかな曲線＋淡いグラデーションへ変更
6. 未入力の未来月を0円として描画せず、最新データ月で停止
7. 月をタップすると売上・利益・利益率を表示
8. キャッシュ名を更新し、iPhoneで旧版が残りにくいよう調整

【更新方法】
ZIPを展開し、GitHubの dashboard-v4 リポジトリ内の既存ファイルを、
このフォルダ内のファイルですべて上書きしてください。

更新後、iPhoneで古い表示が残る場合：
Safariでページを開き直す → 数秒待つ → 再読み込みしてください。
ホーム画面版も一度終了して再度開いてください。

※既存データはブラウザ内に保存されているため、通常はファイルを上書きしても消えません。
念のため「データ」タブから完全バックアップを保存してから更新してください。
