更新方法
1. 現在のアプリで「データ」→「完全バックアップ」を保存
2. ZIPを展開
3. フォルダ内の全ファイルをGitHubの既存ファイルへ上書き
4. GitHub Pages反映後、Safariを再読み込み
5. 古い表示が残る場合はホーム画面アプリを完全終了して開き直す

既存データは同じlocalStorageキーを使用するため、そのまま引き継がれます。
