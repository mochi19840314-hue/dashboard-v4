横浜都筑どうぶつ病院 KPI Dashboard v5.0 Phase 2

更新内容
- 年間グラフのスマートフォン表示不具合を修正
- 1月から最新入力月までを画面幅内に正しく表示
- 未入力の未来月は描画しない
- 赤字月を0円に丸めず、マイナス利益として正しく描画
- 月のタップで売上・利益・利益率を表示
- 院長ダッシュボードを追加
- 推定年収（事業所得目安）を追加
- 年末売上予測・年末利益予測を追加
- 過去最高月商を追加
- 任意の目標年収入力と達成率表示を追加

更新方法
1. 現在のアプリで「データ」→「完全バックアップ」を保存
2. ZIPを展開
3. dashboard-v5-phase2 フォルダ内の全ファイルをGitHubの既存ファイルへ上書き
4. GitHub Pagesの反映後、Safariを再読み込み
5. 古い表示が残る場合はホーム画面アプリを一度終了して再度開く

既存データは同じlocalStorageキーを使用するため、そのまま引き継がれます。
